Chandrayaan-2 OHRC Image Enhancement

Harsh lunar lighting and Permanently Shadowed Regions made Chandrayaan-2’s OHRC images noisy, low-contrast, and hard to analyze. We developed an advanced enhancement pipeline using preprocessing, contrast stretching, histogram equalization, denoising, and adaptive illumination correction. This improved clarity by up to 152% in shadowed areas, reduced noise, and enabled more accurate crater detection and geological mapping.

Resort Booking

Traditional rental platforms inspired the need for a user-centric solution. I built an Airbnb Clone that enables seamless property browsing, booking, and management with secure authentication, dynamic search and filtering, and real-time availability tracking. The system integrates responsive UI design, scalable backend APIs, and database-driven storage for users, listings, and reservations. This streamlined the rental experience by providing hosts with easy property management tools and guests with fast, reliable booking functionality.

Android Chat App

Growing concerns over data security and digital surveillance inspired the development of a next-generation Privacy Chat Application. I designed it with end-to-end encryption, anonymous identifiers, disappearing messages, and screenshot detection to ensure maximum confidentiality. Unlike conventional messengers, no chats are stored on the cloud—messages remain only on user devices with customizable auto-deletion timers. The app also integrates anime-style video calls using real-time style transfer, preserving both identity and privacy. This created a secure, user-first communication platform that blends strong encryption with innovative anonymity features.
